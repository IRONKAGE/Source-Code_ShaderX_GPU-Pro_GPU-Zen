
Running
-------
Run V.exe to start the demo.

Press the '~' key to bring up the console. It allows you to enter debug visualization commands to display the various textures used by the system.

System Requirements
-------------------
This demo requires an NVIDIA gpu compute capability 1.3 card to work.
