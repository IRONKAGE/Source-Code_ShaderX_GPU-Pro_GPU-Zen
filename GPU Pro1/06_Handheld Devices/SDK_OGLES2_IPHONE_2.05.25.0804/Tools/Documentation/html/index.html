OGLES2Tools

Overview

OGLES2Tools is a collection of source code to help developers with some common tasks which are frequently used in 3D programming. OGLES2Tools supplies code for mathematical operations, matrix handling, loading 3D models and to optimise geometry. The API specific tools contain code for displaying text and loading textures.

Content

This is a description of the files which compose OGLES2Tools. Not all the files might have been released for your platform so check the file list to know what is available.

PVRTBoneBatch: Group vertices per bones to allow skinning when the maximum number of bones is limited.

PVRTDecompress: Descompress PVRTC texture format.

PVRTFixedPoint: Fast fixed point mathematical functions.

PVRTMatrix: Vector and Matrix functions.

PVRTVector: Vector and Matrix functions that are gradually replacing PVRTMatrix.

PVRTQuaternion: Quaternion functions.

PVRTResourceFile: The tools code for loading files included using FileWrap.

PVRTMisc: Skybox, line plane intersection code, etc...

PVRTModelPOD: Load geometry and animation from a POD file.

PVRTTrans: Transformation and projection functions.

PVRTTriStrip: Geometry optimization using strips.

PVRTVertex.cpp: Vertex order optimisation for 3D acceleration.

PVRTPrint3D: Display text/logos on the screen.

PVRTTexture: Load textures from resources, BMP or PVR files.

PVRTBackground: Create a textured background.

PVRTError: Error codes and tools output debug.

PVRTShadowVol: Tools code for creating shadow volumes.

PVRTString: A string class.

PVRTShader: Code to load vertex, pixel and geometry shaders.

PVRTPFXParser: Code to parse our PFX file format.
