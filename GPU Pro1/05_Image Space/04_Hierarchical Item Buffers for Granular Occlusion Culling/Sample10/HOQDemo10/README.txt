Key mappings:
space bar:  Switches between render paths
F4:         Toggles animation on/off
+:          Increases the number of shadow casters
-:          Decreases the number of shadow casters
h:          Hides shadow volumes
g:          Loads the shadow casters' transformation matrices from a file and 
            circles through a list of light configurations (only works
            when the animation is running).
m:          Save the shadow casters' transformation matrices to a file
n:          Load the shadow casters' transformation matrices from a file
