------------------
Keyboard Controls:
------------------

H:		Help (shows this file)

LeftArrow:	Look Left
RightArrow: 	Look Right
UpArrow: 	Go Forward
DownArrow: 	Go Backward
+: 		Go Up
-: 		Go Down
PageUp: 	Look Up
PageDown: 	Look Down
*:		Increase Tessellation (decrease fMaxRegionSpan)
/:		Decrease Tessellation (increase fMaxRegionSpan)

W:		Toggle Wireframe
F3:		Toggle FullScreen Mode

ESC:		Reset Settings
SpaceBar:	Render Frame (software mode only)

---------------
Mouse Controls:
---------------

When pressing the left mouse button, use the mouse to look up/down/left/right
