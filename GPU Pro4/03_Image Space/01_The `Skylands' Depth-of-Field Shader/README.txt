Sample code for a Near and Far Field Depth of Field Shader from GPU Pro 4

Code by Morgan McGuire, Mike Bukowski, Padraic Hennessy, and Brian Osman.

This program compiles for Win32, Win64, and OS X 32-bit and 
requires OpenGL 3.0 or later.  It was only tested on NVIDIA GPUs.

This program uses G3D 9.00 beta as support code, checked out from SVN top of 
tree on 2012-09-10 https://g3d.svn.sourceforge.net/svnroot/g3d.  
See http://g3d.sf.net
