pChannels A list of the wanted channels, e.g. {'x', 'y', 0}
Description:
Reduce the number of dimensions in 'data' using the requested channel array. The array should have a maximum length of 4 or be null terminated if less channels are wanted. Supported elements are 'x','y','z' and 'w'. They must be defined in lower case. It is also possible to negate an element, e.g. {'x','y', -'z'}.
